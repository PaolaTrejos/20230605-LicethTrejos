- git init: Sirve para inicializar un repositorio de forma local.
- git commit: Sirve para confirmar los cambios que vamos a subir al repositorio remoto.
- git status: Sirve para revisar el estado de los cambios que hemos realizado.
- git push: Sirve para cargar los cambios realizados de forma local a nuestro repositorio remoto.
- git pull: Sirve para traer contenido de un repositorio remoto a uno local.
